# CursoIA-Cpp

Repositorio con ejercicios y ejemplos del curso de IA en C++.

## Compilación
Usamos CMake como sistema de construcción.

```bash
mkdir -p build && cd build
cmake ..
cmake --build . --config Release
```

En macOS, asegúrate de tener instaladas las Command Line Tools (Xcode) y CMake (`brew install cmake`).
